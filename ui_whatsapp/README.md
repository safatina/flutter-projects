## UI WhatsApp 
Aplikasi ini merupakan tampilan (UI) dari WhatsApp yang dibuat menggunakan Flutter. Fokus utama project ini adalah meniru tampilan antarmuka WhatsApp secara responsif.

## Fitur
- Tampilan chat list
- Status UI
- Call UI
- Navigasi tab (Chat, Status, Calls, community)

## Teknologi
- Flutter
- Dart

## Cara Menjalankan
flutter pub get
flutter run
