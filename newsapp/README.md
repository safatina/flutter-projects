# News App
Aplikasi berita berbasis Flutter yang mengambil data dari API dan menampilkan daftar berita terbaru.

# Fitur
- Fetch berita dari API
- List berita
- Detail berita
- Navigasi antar halaman
- search berita
- bookmark
- share berita
- filter per kategori

# Teknologi
- Flutter
- Dart
- REST API

## 🚀 Cara Menjalankan
```bash
flutter pub get
flutter run
