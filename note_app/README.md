# Note App
Aplikasi catatan sederhana berbasis Flutter yang memungkinkan pengguna untuk menambah, melihat, dan mengelola catatan.

# Fitur
- Tambah catatan
- Edit catatan
- Delete catatan
- Tampilan list catatan
- Search
- Sorting (Newest First, Oldest First, A to Z, Z to A, Most Important)

# Teknologi
- Flutter
- Dart

# Cara Menjalankan
```bash
flutter pub get
flutter run
