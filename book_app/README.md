# 📚 Book App
Aplikasi katalog buku berbasis Flutter untuk menampilkan daftar buku dan detailnya.

## ✨ Fitur
- List buku
- Detail buku (rating, jumlah pages, language, descriprion)
- filter berdasarkan kategori 

## 🛠️ Teknologi
- Flutter
- Dart

## 🚀 Cara Menjalankan
```bash
flutter pub get
flutter run
